Linux Release Roundup #21.35: Ubuntu 20.04.3, OpenShot 2.6, MX Linux 21, and More New Releases
======

In the Linux Release Roundup series, we summarize the new application and distribution versions release in the last few days. This keeps you informed of the latest developments in the Linux world.

### Ubuntu 20.04.3 LTS Released

The third point release for Ubuntu 20.04 LTS has finally arrived right around the time of Ubuntu 22.04 LTS release date reveal.

It includes Linux Kernel 5.11 and many other updates to packages. To learn more about the release, you can refer to the [release announcement][1].

### OpenShot 2.6 Released

OpenShot 2.6 is a major release for the video editor adding AI and computer vision effects.

You can check out our [original coverage][2] to learn more about the release.

### GNOME 41 Beta Released

GNOME 41 is the next version of the desktop environment with some significant changes but not a big release.

The beta release is already available for public testing. So, go through our [GNOME 41 beta coverage][3] before proceeding, if interested.

### MX Linux 21 Beta Released

MX Linux 21 series includes an upcoming KDE and Fluxbox edition. The beta versions are finally available for testing.

Take a look at MX Linux’s [official blog][4] to catch up with the latest available options.

### Voyager Live 11 Released

A Debian-based desktop Linux distribution with a customized GNOME experience.

The new version is based on the latest Debian 11 and updated GNOME 3.38 along with other latest packages.

You can read the [original announcement][5] to learn more.

### Clonezilla 2.7.3 Released

Clonezilla 2.7.3 is not a major release but it has received a kernel update along with numerous improvements.

The [official release notes][6] should inform you more about the release.
