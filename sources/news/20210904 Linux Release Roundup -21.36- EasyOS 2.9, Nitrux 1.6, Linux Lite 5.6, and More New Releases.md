Linux Release Roundup #21.36: EasyOS 2.9, Nitrux 1.6, Linux Lite 5.6, and More New Releases
======

In the Linux Release Roundup series, we summarize the new application and distribution versions release in the last few days. This keeps you informed of the latest developments in the Linux world.

### Nitrux 1.6 Released

Nitrux is a Debian-based Linux distribution. It comes loaded with the latest [Linux Kernel 5.14][1] and more improvements.

You can check out its [official release announcement][2] to learn more.

### EasyOS 2.9 Released

A minimal Linux distribution inspired from Puppy Linux. EasyOS 2.9 introduces several package updates and fixes issues found in the previous version.

You can go through its [release notes][3] to explore all the technical changes.

### Linux Lite 5.6 Released

Linux Lite 5.6 may not be a major release in terms of technical changes but they have moved to a “pay what you want” model just like elementaryOS.

Read our [original coverage][4] to learn more about it.

### Linux From Scratch 11.0 Released

Linux From Scratch 11.0 is a major version bump where you can notice Linux Kernel upgrade, and some system changes.

To check out the list of detailed changes, I suggest you check the [official news page][5].

### EndeavourOS 2021.08.27 Released

EndeavourOS’ latest version includes an improved installer and a new app. You should check out the [official release announcement][6] for more details.
