Linux Release Roundup #21.33: elementary OS 6, Thunderbird 91, and More New Releases
======

In the Linux Release Roundup series, we summarize the new application and distribution versions release in the last few days. This keeps you informed of the latest developments in the Linux world.

### elementary OS 6 Released

elementary OS 6 is the biggest release of this week which introduced some massive feature additions and improvements to the elementary OS platform.

You can read our [release co][1][v][1][erage][1] and the [feature list of elementary OS 6][2] to explore more about it.

### Thunderbird 91 Released

Thunderbird is not just another usual release but a major upgrade that introduces CalDAV support and several user interface improvements.

Read [our coverage][3] to know why it might help Thunderbird reclaim its position as the default Linux mail client.

### KDE Gear 21.08 Released

KDE Gear 21.08 is a significant upgrade that introduces several new features for Dolphin file manager, Konsole terminal, Kdenlive video editor, and others.

You can read all the details in their [official announcement post][4].

### Firefox 91 Released

Firefox 91 is mostly a huge bug fix release that also includes some new improvements.

You can choose to take a look at the [release notes][5] to find out more.

### Elive 3.8.22 Beta Released

Elive is a Debian-based distribution that features the Enlightenment desktop.

The latest beta release updates the applications and switches the IRC chat server to the Libera server. You may refer to the [release notes][6] to know more.
