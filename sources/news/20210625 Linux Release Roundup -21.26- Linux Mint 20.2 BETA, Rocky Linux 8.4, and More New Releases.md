Linux Release Roundup #21.26: Linux Mint 20.2 BETA, Rocky Linux 8.4, and More New Releases
======

In the Linux Release Roundup series, we summarize the new application and distribution versions release in the last few days. This keeps you informed with the latest developments in the Linux world.

### Rocky Linux 8.4 Released

Rocky Linux 8.4 is the first stable release that aims to replace CentOS on production servers.

You can learn more about it in our [original coverage][1].

### SUSE Linux Enterprise 15 SP3 Released

The third service pack of the SUSE’s enterprise offering is here that announces the full compatibility with openSUSE 15.3 release.

You can check out the [official release notes][1] for more information.

### DVDStyler 3.2 Released

A DVD authoring tool got a new update that improves the user interface, adds/tweaks a few features along with some bug fixes.

You can learn more about it in their [official news post][2].

### IPFire 2.25 Core 157 Released

IPFire’s minor release involves the work being done to get rid of Python 2.0 code and migrate to Python 3.

With the next release, the transition should complete. To check out the details of this release, check the [official blog post][3].

### Linux Mint 20.2 Beta Released

Linux Mint 20.2 beta is an exciting release that introduces new applications and features.

You can explore more about it in our [original coverage][4].
