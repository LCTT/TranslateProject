Ubuntu 20.10 到达生命终点，该升级啦！
======

Ubuntu 20.10 (Groovy Gorilla) 今天停止支持啦(2021年6月22日)。它是一个非长期支持发行版本(non-LTS)[带来了一些新特性][1].

通常，非长期支持发行版维护9个月，所以20.10生命周期结束意味着Ubuntu 20.10用户不会再收到安全和维护更新。

你也会错过已安装应用的更新，安装新软件也会面临问题[使用命令行][2]。没有了日常的sources.list维护，使用软件中心也将是个问题（也不推荐）。

支持周期的结束针对所有的其他Ubuntu特色版本像Kubuntu, Lubuntu, MATE等。

但是像Linux Mint和elementary OS发行版，它们不依赖于非长期支持发行版，你不必有任何担心。

最好使用下面的命令在终端[检查下你安装的Ubuntu版本][3]

```
lsb_release -a
```

### 升级到Ubuntu 21.04 (Hirsute Hippo):正是时候!

毫无疑问，你需要升级到ubuntu21.04才能获得最新的更新和安全改进。

[Ubuntu 21.10 版本][4]将在几个月后发布，所以你也可以到时升级到21.10。

现在，你可以跟着我们的[Ubuntu升级][5]教程来开始升级。


我推荐你备份一下，以防升级过程中出现问题。


如果你想重新安装，也可以。

### 还在用Ubuntu 20.10?

技术上来说，你可以继续使用不受支持的版本，如果你不想有安全问题出现，建议还是升级到最新的版本，比如Ubuntu 21.04。


Ubuntu 21.04将会提供支持到2022年1月，升级前你可能想看下[ Ubuntu 21.04特征][6].
