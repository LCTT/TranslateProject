Linux Release Roundup #21.25: ActivityWatch 0.11, elementaryOS 6 Beta 2, and More New Releases
======

In the Linux Release Roundup series, we summarize the new application and distribution versions release in the last few days. This keeps you informed with the latest developments in the Linux world.

### PDF Mix Tool 1.0.1 Released

PDF Mix Tool is an incredibly useful tool. With the latest release, it lets you edit metadata and adds support for Qt 6.

To learn more about the release, you can refer to our [original coverage][1].

### elementary OS 6 Beta 2 Released

elementary OS 6 is one of the [most anticipated distros of 2021][2]. The beta 2 release has exciting improvements that you can try.

Have a look at [our article][3] on it to know more about it along with the instructions to download it.

### ActivityWatch 0.11 Released

ActivityWatch is a unique time tracker available for Linux and other platforms. The latest release improves the user interface with more features.

You can read more about it in our [original coverage][4].

### Regolith Linux 1.6 Released

Regolith Linux is an Ubuntu-based distro that comes with a new look and several technical improvements.

You can learn more about by heading to its [official website][5].

### Rocky Linux 8.4 RC 1 Released

Rocky Linux 8.4 first release candidate is here with memory management improvements and several other changes.

If you are curious about all the technical details and want to test it out, you can check the [release notes][6].
