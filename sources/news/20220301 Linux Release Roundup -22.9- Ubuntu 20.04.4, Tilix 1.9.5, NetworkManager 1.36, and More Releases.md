In the Linux Release Roundup series, we summarize the new distribution and application version releases in the past week. This keeps you informed of the latest developments in the Linux world

### Ubuntu 20.04.4 LTS Released

Ubuntu 20.04 LTS received its fourth point release with support for newer hardware like Intel’s Alder Lake processors and a few Foxconn wireless modems.

You can read the [official announcement][1] to learn more.

### Tilix 1.9.5 Released

Tilix is one of the most popular terminal emulators. However, due to lack of maintainers, they only managed to offer a few new things with the new update after a year.

You can explore more about it on its [GitHub page][2].

### GIMP 2.99.10 Development Released

With GIMP 3.0 release soon (no ETA), GIMP 2.99.10 development version has arrived with numerous improvements and some new additions.

To learn more about the release, you can refer to the [official blog post][3].

### NetworkManager 1.36 Released

NetworkManager 1.36 works on improving the way IP configuration is handled, and fixed some performance issues as well.

It is one of the major releases after six weeks of its last release. The [official blog post][4] should describe more about it.

### TeXworks 0.6.7 Released

TeXworks is one of the best LaTeX editors available. The new release adds some minor features and improves the dark mode among other changes.

The [official announcement news][5] should provide you with more technical details about the changes.
