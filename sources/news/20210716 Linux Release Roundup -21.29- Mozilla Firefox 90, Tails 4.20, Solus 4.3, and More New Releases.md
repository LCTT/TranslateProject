Linux Release Roundup #21.29: Mozilla Firefox 90, Tails 4.20, Solus 4.3, and More New Releases
======

In the Linux Release Roundup series, we summarize the new application and distribution versions release in the last few days. This keeps you informed with the latest developments in the Linux world.

### Tails 4.20 Released

Tails 4.20 is an interesting release that introduces a Tor connection assistant that makes it easy for anyone to setup a Tor connection.

Along with that, several apps like [OnionShare][1] have been updated to its latest versions.

You can learn more in their [official news announcement][2].

### Solus 4.3 Released

Solus came up with their third minor release with essential improvements and upgrades.

You can check out our [coverage][3] to learn more about it.

### Firefox 90 Released

Mozilla Firefox’s version 90 is not a big release but comes baked in with a new version of SmartBlock feature to block scripts tracking you and the ability to manage HTTPS only exceptions.

Check out the [official release notes][4] to know more about it.

### digiKam 7.3 Released

digiKam is an open-source photo management tool. With the latest update, it introduces a new EXIF tool, improved conversion features, and more.

You should look at the [official announcement][5] to explore more about it.

### Box64 Emulator Released

Box64 emulator is a new addition that lets you run 64-bit Linux programs on an ARM system.

You can explore more about it in [our coverage][6].
