Linux Release Roundup #21.28: Linux Mint 20.2, KDE Gear 21.04.3, Tor Browser 10.5, and More New Releases
======

In the Linux Release Roundup series, we summarize the new application and distribution versions release in the last few days. This keeps you informed with the latest developments in the Linux world.

### Linux Mint 20.2 Released

Linux Mint 20.2 final release is here with significant upgrade and improvements.

You can learn more about the changes in our [news coverage][1].

### Nextcloud Hub 22 Released

Nextcloud is a collaboration platform that you can self-host to manage teams and your work. With the latest release, the collaboration gets easier along with a few exciting features.

For more information you can refer to our [original coverage][2] about this release.

### Tor Browser 10.5 Released

Tor Browser announced their latest update that introduces Wayland support along with some UX improvements.

You can check out their [official blog post][3] to learn more about it.

### KDE Plasma 5.22.3 Released

KDE Plasma’s next maintenance release is here with several bug fixes.

To go through the technical details, refer to the [official announcement][4].

### KDE Gear 21.04.3 Released

KDE’s suite of applications received a bunch of bug fixes with their third maintenance release.

To get all the details, refer to the [official announcement post][5].

### Darktable 3.6 Released

Darktable, the open-source RAW image editor received a major update with reworked import module, a new RGB color balance module, and more.

You can learn more about all the feature changes in their [blog post][6].

### Pop!_OS 21.04 Released

Pop!_OS 21.04 is available to download. It debuts with the new COSMIC desktop environment, based on GNOME.

Check out our [news coverage][7] to learn more.
