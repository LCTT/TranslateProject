Linux Release Roundup #22.8: Slax 11.2, OBS Studio 27.2, Kali Linux 2022.1, and More Releases
======

In the Linux Release Roundup series, we summarize the new distribution and application version releases in the past week. This keeps you informed of the latest developments in the Linux world.

### Slax 11.2 Released

Slax 11.2 is a major update after 2 years of any previous significant development activity.

This release is based on Debian 11.2 Bullseye and adds/removes a couple of packages. You can learn more about it in [our coverage][1].

### OBS Studio 27.2 Released

OBS Studio 27.2 introduced several Linux-focused improvements and has made it easier with official Flatpak support.

Refer to our [original coverage][2] to know more about the release.

### Kali Linux 2022.1 Released

Kali Linux introduced some major visual changes, with a new plan to implement them with the first update every year.

There’s also a new “Everything ISO” available. Take a look at our [Kali Linux 2022.][3][1][3] [release article][3] to explore more.

### AV Linux MX-21 Released

AV Linux is a media production-focused distribution, originally based on MX 21 and Debian 11 Bullseye.

You can check out the [official release announcement][4] for technical details.

### Bottles 2022.2.14 Released

Bottles help you run Windows software on Linux. With its latest version, you get installers to quickly set up supported Windows applications without much of a hassle.

Take a look at [our coverage][5] to know other improvements that come along with the release.
