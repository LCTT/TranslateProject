Linux Release Roundup #21.50: Zorin OS 16 Lite, Tails 4.25, FreeBSD 12.3, and More Releases
======

In the Linux Release Roundup series, we summarize the new distribution and application version releases in the past week. This keeps you informed of the latest developments in the Linux world.

### Zorin OS 16 Lite Released

Zorin OS 16 Lite is a lightweight Linux distribution tailored to run on older computers. It features the latest Xfce 4.16 desktop along with a revamped look that makes Xfce pretty.

You can find more details in [our coverage][1].

### Kali Linux 2021.4 Released

With the latest Kali Linux release, they have improved the support for ARM devices and Raspberry Pi Zero 2.

To explore all the technical changes, you can refer to its [official changelog][2].

### FreeBSD 12.3 Released

FreeBSD 12.3 is a minor update that involves bug fixes, and some userland updates.

For more details, you can check out its [official blog post][3].

### Tails 4.25 Released

Tails 4.25 improves the ability to back up the persistent storage to another USB stick or an external drive.

If you are worried about online privacy and want total anonymity, you can try the latest release and explore all the details in their [blog post][4].

### EndeavourOS 2021.4 Released

The latest EndeavourOS release updates several applications and improves the experience with NVIDIA graphics.

The [official announcement post][5] should shed more light on it.
