Linux Release Roundup #21.38: RetroArch on Steam, Kali Linux 2021.3, and More New Releases
======

In the Linux Release Roundup series, we highlight the new distribution and application version releases in the past week. This keeps you informed of the latest developments in the Linux world.

### Kali Linux 2021.3

The cybersecurity-focused Debian-based distro, Kali Linux, has received major upgrades in its latest release. This includes improvements to OpenSSL, Virtualization, and an upgrade to Kali Tools.

In addition, the TicHunter Pro – their first NetHunter smartwatch – has also been announced.

You can learn more by checking out our [original coverage.][1]

### RetroArch

RetroArch, the popular front-end for emulators, is now available on **Steam** exclusively for Linux and Windows users. About 10 cores or emulators are included as of now and can be downloaded as DLCs.

Check out our [original coverage][2] to know more.

### Darktable 3.6.1

The free and open-source RAW image editor Darktable’s latest release fixes numerous bugs and extends its base support for more cameras, including a noise profile.

Their [official changelog][3] contains the details and download files.

### Gnome Subtitles 1.7

Gnome Subtitles finally received a significant upgrade after two quiet years. It is a handy and open-source subtitle editor that offers features like subtitle translation and integrated video previewing.

This release contains important bug fixes, an upgrade to GStreamer media playback, and improved audio and video support.

Do check out their [official release notes][4] for detailed information.

### ExTiX 21.9

ExtiX is a Deepin based Linux OS from Sweden. There are have been quite a number of new updates and features in its latest release. This includes the use of kernel 5.14.2-exton, the Reborn version of the Deepin installer, and the Cactus Package Manager.

In-depth details about the release can be found on their [official website.][5]
