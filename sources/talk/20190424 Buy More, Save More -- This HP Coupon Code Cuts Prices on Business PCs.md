Buy More, Save More -- This HP Coupon Code Cuts Prices on Business PCs
======

![HP][1]

If you're looking to buy a new business PC, HP is currently offering a "[buy more, save more][2]" discount that gets activated at 3 different price points, and using a special code.

Configure your new PC to the $599+ price point and apply code [BMSM60][2] to save $60. Or configure to $1,299+ and use code [BMSM150][3] to save $150. Configure to $1,999+ and use code [BMSM350][4] for a whopping $350 off your new PC. Click the codes just mentioned to learn more, or [click right here][2] to review available models, specs, and configurations.
