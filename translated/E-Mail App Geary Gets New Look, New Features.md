E-Mail应用“Geary”更新，新外观，新功能
================================================================================
![](http://www.omgubuntu.co.uk/wp-content/uploads/2013/10/Screen-Shot-2013-10-04-at-14.30.29.png)
*Geary的新貌：0.4版本已经可供下载*

在[拍照工具"Shotwell"版本更新][1]的网页中，有消息提到那个开源的E-Mail桌面应用“Geary“有更新了。

“Geary”由Yorba开发，在[众筹集资10万美元失败][2]之后，其开发团队于四月强势回归。

我喜欢这个轻量级的应用已经很久了，这次升级它又提供了大量新特性，其中一些已经被期待很久了。

Geary新提供**搜索、自动保存草稿以及内置安全工具**功能，其中搜索功能不能跨帐号执行，安全工具可以检测对方地址的恶意内容。

它那个一直是“先显示邮件列表，再显示邮件内容”的界面也有所改变。

侧边栏能显示**文件夹内未读邮件数量**，并使用了新图标； 工具栏也使用了新的菜单键。

![](http://www.omgubuntu.co.uk/wp-content/uploads/2013/10/Screen-Shot-2013-10-04-at-14.25.09.png)
*Geary新的菜单键和侧栏图标*

邮件列表栏不再默认显示星号标记和阅读标记，仅在鼠标移到邮件上时才出现。当用户给某邮件加星标时，它们还是会显示出来的。

![](http://www.omgubuntu.co.uk/wp-content/uploads/2013/10/Screen-Shot-2013-10-04-at-14.24.23.png)
*从上到下为：未读＆加星标；己读＆未加星标；未读*

**Ubuntu上安装Geary 0.4**

如果你在用**Ubuntu 12.10或13.04**，你可以[从Yorba PPA下载Geary 0.4][3]。

打开*Terminal终端*并输入命令：

    sudo add-apt-repository ppa:yorba/ppa
    
    sudo apt-get update && sudo apt-get install geary

Ubuntu 13.10 用户可以在[Ubuntu软件中心][4]找到它。
