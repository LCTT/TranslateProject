龙芯正在为 LoongArch Linux 笔记本电脑做准备
======
这家中国制造商正在开发基于 LoongArch 处理器的 Linux 笔记本电脑系列。

![Loongson is Getting Ready for LoongArch Linux Laptops][1]

不要将它与 Arch Linux 混淆 :)

LoongArch 是龙芯科技的 CPU 架构。

近期，龙芯在Linux Kernel 5.19中加入了对 LoongArch CPU 架构的初步支持。

如果您好奇，LoongArch 与 RISC-V 具有相似的属性。您也可以将其称为新的 RISC ISA（指令集架构）。

想要更多技术细节？Linux 内核的文档是您的朋友。

虽然它并不流行，但始终欢迎新的 CPU 架构支持，并通过新硬件开辟各种机会。

中国制造商为 Linux 笔记本电脑做准备

负责 LoongArch CPU 架构的计算机硬件制造商龙芯科技似乎正在准备具有相同功能的新 Linux 笔记本电脑。

[LoongArch] 最新款龙芯笔记本开箱——龙芯3A5000

这意味着什么：您很快就会看到运行开箱即用的 Linux 的 LoongArch 笔记本电脑。

当然，考虑到它作为中国供应商，它不太可能随处可见。但你永远不知道，对吧？

联想和华为是中国最著名的 Linux 笔记本电脑制造商之一。考虑到 Linux 笔记本电脑市场并不拥挤，龙芯推出的产品听起来不错。

2021 年购买 Linux 笔记本电脑的 13 个地方

**龙芯向 Linux 添加驱动程序：**他们正在向 Linux 内核添加代码，即Phoronix发现的基于 ACPI 的“龙芯笔记本电脑”驱动程序的形式。这可能是即将发布的Linux 6.0 版本的一部分。

T代码添加中还提到了一些源自 Thinkpad 的 ACPI 配置，这应该不足为奇。

尽早加入 RISC-V 竞赛：考虑到 RISC-V 计算机有望成为计算领域的下一个重大飞跃，我认为龙芯希望在该领域展开竞争，为现实世界准备类似的 CPU 架构。

龙芯的笔记本电脑或其性能并不出名。因此，看看它如何很快展开将会很有趣。
