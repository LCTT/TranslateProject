增加了新功能和修复1000bug的VLC 2.1发布
================================================================================
VLC 是一个强大的,多功能的先进媒体播放器,它包含大量的功能并且支持绝大多数已存在的解码器和视频格式,允许无障碍的内容摘要体验.

**VLC 2.1 "Rincewind"** 已经[发布][1], 它带来了显著的修复,优化和改进,包括:


    重写音频核心改进音量处理
    重修音频模块适配新的音频核心
    4k视频支持
    通过VDPAU硬件解码
    支持不完整的MP4文件
    增加支持MKV,MJPEG和AVI流
    增强蓝光输入
    加强MKV seeking
    改进MKV文件metadata和字幕支持


![](http://iloveubuntu.net/pictures_me/vlc%202.1%20stable%20saucy.png)


VLC 2.1 带来了大约 **1000 bug 修复** , 从而坚实的加强便利的媒体播放器,相对于之前的版本它在一个新的高度增强了底层稳定性,速度和可靠性.


我们怎样 **安装** VLC 2.1?
加上下面的 **official** PPA (Ubuntu 13.10)

    sudo add-apt-repository ppa:videolan/stable-daily
    
    sudo apt-get update
    sudo apt-get install vlc
